# Devireen Enterprise Blueprint

## Vision

Build a modern B2B-first procurement platform for Devireen Enterprise
using Next.js, Tailwind CSS, Supabase and Vercel. The website serves
businesses (60% of revenue) and retail customers (40%) while being
architected to evolve into a full ecommerce platform.

## Business Summary

- 5+ years in business
- Nairobi CBD
- 2 branches
- 500+ products
- Nationwide delivery
- Public pricing
- Quote-first purchasing workflow

## Objectives

- Establish trust
- Drive quotation requests
- Improve SEO
- Provide an exceptional mobile experience
- Create a reusable commerce platform for Blackpool Industry

## Technology

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Supabase (Postgres, Auth, Storage)
- Vercel hosting

## Customer Journey

Home → Search/Browse → Product → Quote Cart → Customer Details →
WhatsApp → Sales Call → Invoice → Paybill → Delivery

## Engineering Rules

- TypeScript only
- App Router only
- Server Components by default
- Reusable components
- Mobile-first
- SEO mandatory
- Accessibility mandatory
- No placeholder production content
- No duplicated code
- Performance over visual effects
