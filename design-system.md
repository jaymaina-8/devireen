# Design System

## Brand

Professional, modern, trustworthy, procurement-focused.

## Colors

Primary: Blue Background: White Surface: Light gray Text: Dark gray
Success: Green Warning: Orange Error: Red

## Typography

Clear sans-serif hierarchy. Consistent spacing scale.

## Components

- Buttons
- Inputs
- Search bar
- Product cards
- Category cards
- Quote cart
- Navigation
- Footer
- Badges
- Breadcrumbs
- Pagination
- Empty states
- Loading skeletons

## Layout

12-column responsive grid. Max-width container. 8px spacing system.

## UX Rules

- Search always visible.
- Quote cart persistent.
- Clear CTA hierarchy.
- Large touch targets.
- Responsive from mobile first.

## Accessibility

- Keyboard navigation
- Semantic HTML
- Proper contrast
- Alt text
- Visible focus states

## Performance

- Optimize images
- Lazy loading
- Minimal JS
- Server Components first

## Animation

Subtle only. Never delay user interaction.
